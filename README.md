TimeLens is a lightweight browser extension that detects UTC timestamps on webpages and converts them to the user's local timezone.

Designed for developers, DevOps engineers, and anyone working with logs or distributed systems.

---

## Features

- Automatic detection of common UTC timestamp formats
- Conversion to system local timezone
- Hover tooltip for non-intrusive display
- Manual conversion of selected text
- Copy converted value
- Configurable timezone override
- No external API calls (local processing only)

---

## Supported Formats (Examples)

- `2026-02-19T12:00:00Z`
- `2026-02-19 12:00:00 UTC`
- `Feb 19 12:00 UTC`

---

## Installation (Manual)

This extension is distributed via source and must be loaded in Developer Mode.

1. Download the ZIP
2. Extract
3. Go to chrome://extensions
4. Enable Developer Mode
5. Load unpacked
